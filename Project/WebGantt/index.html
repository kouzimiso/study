スケジュール（暗号化）

ログインしてください

Classic のトークンを使ってください（Fine-grained は Gist API で 403 になります）。 このリンク を開くと gist スコープが選択済みで開きます。期限を選び「Generate token」→ ghp_ で始まる文字列を貼り付け。 （手動の場合: Settings → Developer settings → Tokens (classic) → gist だけチェック）

パスフレーズを入力

スケジュールの暗号化に使う合言葉です。ログインパスワードとは別物です。

⚠ パスフレーズを忘れると復号できません（サーバーにも鍵はありません）。控えを安全に保管してください。
